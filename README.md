# Tulpa

> フリーランスエージェントの事務業務を、スマートに。

---

フリーランスエージェントの担当者は「いい会社にいいエンジニアを紹介したい」という思いで仕事をしています。

Tulpaは、契約管理・稼働記録・請求処理・支払通知をひとつにまとめた、オープンソースのフリーランスエージェント管理ツールです。AIマッチング機能を備えています。

---

## 主な機能

### エンジニア向け

- スマホから稼働記録を入力・提出。月末のExcel作業をなくす
- 契約・支払通知・スキルシートが一か所に。
- 年間カレンダーで稼働ペースと休暇余裕を把握

### エージェント向け

- 月次稼働報告の確認・差戻しフロー
- 契約条件と契約当事者の組み合わせを含む契約管理
- 毎月の請求書・支払通知の自動生成
- エンジニアCRM・クライアントCRM（営業活動・次アクション管理）
- AIによるスキルシート×案件要件のマッチングサマリー生成

### クライアント向け

- 案件要件の登録・更新・公開・非公開設定（エージェントによる登録代行も可能）
- 案件要件に対するエンジニアのマッチングサマリー確認
- 稼働報告のオンライン確認・差戻し
- 契約内容・請求書のオンライン確認

## 技術スタック

レンタルサーバーでも動作する軽量設計です。

| カテゴリ | 技術 |
|--------|------|
| バックエンド | PHP 8.2+ / Laravel 12 |
| フロントエンド | Alpine.js + htmx（ビルド不要）/ Tailwind CSS |
| データベース | MySQL 8.0+ / MariaDB 10.6+ |
| オプション | AIマッチング / Slack API（通知連携）|
| モバイル | PWA（スマホのホーム画面から起動可能）|

## インストール

```bash
git clone https://github.com/yama/tulpa.git
cd tulpa
composer install --no-dev
cp .env.example .env
php artisan key:generate
php artisan migrate
```

詳細なセットアップ手順は今後追加予定です。
