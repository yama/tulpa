# Repetition Review

## Signals To Check

- A word appears multiple times in one paragraph and each use carries the same meaning.
- Adjacent sentences repeat the same grammatical frame.
- A replacement introduced a new hotspot word elsewhere in the paragraph or section.
- A heading and the first sentence repeat the same phrase without adding context.

## Intentional Repetition

Keep repetition when it does one of these jobs:

- Defines or reinforces a key term
- Holds together a section's main claim
- Preserves legal, product, or domain precision
- Makes a contrast legible across several sentences

## Accidental Repetition

Rewrite repetition when it does one of these:

- Fills space without sharpening the point
- Repeats the same abstract noun instead of moving the thought forward
- Makes the paragraph sound mechanically generated
- Draws attention away from the actual argument

## Rewrite Order

1. Keep the anchor term.
2. Rewrite the supporting occurrence closest to it.
3. Re-read the full paragraph.
4. Re-scan the section for new hotspot words created by the edit.

## Safe Rewrite Moves

- Replace one repeated noun with a concrete explanation
- Replace one repeated verb with the effect or result
- Merge two sentences that restate the same idea
- Split the paragraph if one topic sentence is carrying too many sub-points
- Remove filler phrases before reaching for synonyms

## Unsafe Rewrite Moves

- Replacing a defined term with a near-synonym that changes meaning
- Using pronouns when multiple subjects are in play
- Varying wording inside tables or terminology lists that should stay aligned
- Rewriting every repeated word in a paragraph and erasing emphasis
